# Project Title
An inefficient implementation of the HCI selection algorithm for use with PySCF.

### Prerequisites
- [Python 3.10+](https://www.python.org/)
- [NumPy](https://numpy.org/)
- [PySCF](https://pyscf.org/)

### Installation
1. Clone the repo: `git clone https://github.com/osbritton1/hci.git <my-folder-name>`
2. Change directories: `cd <my-folder-name>`
3. Activate a conda environment: `conda activate <my-env>`
4. Install dependencies: `python3 -m pip install .`
