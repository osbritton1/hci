Assembles all diagonal matrix elements for configurations contained in hcivec; used for the Davidson preconditioner.

Parameters:

  • hcivec (HCIVec) – Reference HCIVec tagged with list of ranks describing the current configuration space

  • config_info (ConfigInfo) – ConfigInfo instance supplying ranking tables and key parameters of the system

  • h1e (HCore) – The core Hamiltonian in the MO bases

  • eri_mo (ERITensor) – The electron repulsion integrals in the MO bases

Raises:

HCIVec.RankNotSetError – rank attribute of hcivec is missing, which is needed to define the current configuration space.

Returns:

The diagonal matrix elements of the Hamiltonian in the subspace defined by hcivec.

Return type:

npt.NDArray[np.float64]
