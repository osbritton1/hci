Combinadic ranking

group Combinadic ranking

Implementation of combinadic ranking system for encoding orbital occupancy and excitation lists.

static size_t find_row_index(uint64_t target, const uint64_t *row, size_t norb, size_t nocc)

Helper function for unranking algorithm.

Uses binary search to find the index of the last entry in a row of the ranking table that is less than the value of target.

Parameters:

  • target – [in] Target value used as a reference for the search

  • row – [in] Pointer to the row of the ranking table used for the search

  • norb – [in] Size of orbital space

  • nocc – [in] Number of occupancies; generally equal to \(N_\alpha\) or \(N_\beta\)

static uint64_t rank(const size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

Get the combinatorial rank corresponding to the provided list of occupied orbitals.

Formula:

\[ \sum\limits_{i=1}^{N_\text{occ}}\binom{c_i}{i}, \]
where \(c_i\) is the \(i\)th orbital in the occupancy list sorted in ascending order

Parameters:

  • occ_list – [in] Pointer to an array of length \(N_\text{occ}\) specifying the zero-indexed occupied orbitals in ascending order

  • rank_table – [in] Pointer to ranking table initialized by load_rank_table

  • norb – [in] Size of orbital space

  • nocc – [in] Number of occupancies; generally equal to \(N_\alpha\) or \(N_\beta\)

Returns:

The rank of the specified occupancy list

static void unrank(uint64_t rank, size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

Greedy unranking algorithm for computing an occupancy list given its combinatorial rank.

Starting from the last row in the ranking table (corresponding to the highest occupied orbital), uses find_row_index to find the column index of the entry that does not exceed the value of target, updates the occupancy list accordingly, then subtracts off the ranking table entry from target and repeats.

Parameters:

  • rank – [in] The rank of the occupancy list to be computed

  • occ_list – [out] Pointer to an array of length \(N_\text{occ}\) to store the occupancy list

  • rank_table – [in] Pointer to ranking table initialized by load_rank_table

  • norb – [in] Size of orbital space

  • nocc – [in] Number of occupancies; generally equal to \(N_\alpha\) or \(N_\beta\)

void load_rank_table(uint64_t *table, size_t norb, size_t nocc)

Initializes the \(N_\text{occ}\) by \(N_\text{orb}-N_\text{occ}+1\) table needed to rank and unrank combinations using the combinatorial number system.

The binomial coefficients needed for the encoding the position of the ith electron are stored in row i-1. The entry in position (i, j) is \(\binom{i+j}{i+1}\); when i+j < i+1 (first column), this is defined to be 0.

Parameters:

  • table – [out] Pointer to the uninitialized ranking table; must accommodate \(N_\text{occ}\cdot(N_\text{orb}-N_\text{occ}+1)\) entries

  • norb – [in] Size of orbital space

  • nocc – [in] Number of occupancies; generally equal to \(N_\alpha\) or \(N_\beta\)
