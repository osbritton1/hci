struct HCore
#include <hci_store.h>

A structure that holds pointers to the core Hamiltonian in both the \(\alpha\) MO and \(\beta\) MO bases.

Neither array is expected to be compressed (i.e. both must have all \(N_\text{orb}^2\) entries).

struct DoubleExcEntry
#include <hci_store.h>

A struct that stores the minimal amount of information necessary to compute a double excitation matrix element given the four changing orbitals encoded by rank (referred to as i, j, k, and l in ascending order).

[ik|lj]-[ij|lk] may be recovered by computing negative the sum of ijkl and iljk; all double excitation matrix elements involving these four orbitals may be mapped to one of these three values up to a sign (see get_double_exc_value_from_store).

Public Members

double iljk

[il|jk]-[ik|jl] in chemist’s notation

struct MixedExcEntry
#include <hci_store.h>

A struct to represent mixed excitations.

Because the ERI tensor is real, only excitations with \(i<j\) and \(k<l\) need to be stored; there are no restrictions otherwise because \(i\) and \(j\) index \(\alpha\) MOs while \(k\) and \(l\) index \(\beta\) MOs.

Public Members

uint64_t rank

The rank of the mixed excitation; see rank_mixed_exc

double ijkl

[ij|kl] in chemist’s notation; i and j are \(\alpha\) orbitals, while k and l are \(\beta\) orbitals
