struct DoubleExcitationEntry
#include <hci_store.h>

A struct that stores the minimal amount of information necessary to compute a double excitation matrix element given the four changing orbitals encoded by rank (referred to as i, j, k, and l in ascending order). [ik|lj]-[ij|lk] may be recovered by computing negative the sum of ijkl and iljk; all double excitation matrix elements involving these four orbitals may be mapped to one of these three values up to a sign.

Public Members

uint64_t rank

The combinatorial rank of the list of four changing orbitals; see rank(const size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

double ijkl

[ij|kl]-[il|kj] in chemist’s notation

double iljk

[il|jk]-[ik|jl] in chemist’s notation

void get_rank_table(uint64_t *table, size_t norb, size_t nocc)

Initializes the nocc by (norb-nocc+1) table needed to rank and unrank combinations using the combinatorial number system. The binomial coefficients needed for the encoding the position of the ith electron are stored in row i-1. The entry in position (i, j) is nCr(i+j, i+1); when i+j < i+1 (first column), this is defined to be 0.

Parameters:

  • table – [out] Pointer to the uninitialized ranking table; must be able to accommodate at least nocc rows by norb-nocc+1 columns

  • norb – [in] Number of orbitals

  • nocc – [in] Number of occupancies; generally equal to the number of alpha or beta electrons in the system

uint64_t rank(const size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

Get the combinatorial rank corresponding to the provided list of occupied orbitals.

Formula:

\[ \sum\limits_{i=1}^{N_\text{occ}}\binom{c_i}{i} \]

Parameters:

  • rank_table – [in] Pointer to ranking table initialized by get_rank_table(uint64_t *table, size_t norb, size_t nocc)

  • norb – [in] Number of orbitals

  • nocc – [in] Number of occupancies; generally equal to the number of alpha or beta electrons in the system

Returns:

The rank of the specified occupancy list

Parameters:

  • row – [in] Pointer to the row of the ranking table used for the search

  • norb – [in] Number of orbitals

  • nocc – [in] Number of occupancies; generally equal to the number of alpha or beta electrons in the system

void unrank(uint64_t rank, size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

Greedy unranking algorithm for computing occupancy list given its combinatorial rank. Starting from the last row in the ranking table (corresponding to the highest occupied orbital), uses find_row_index(uint64_t target, uint64_t *row, size_t norb, size_t nocc) to find the column index of the entry that does not exceed the value of target, updates the occupancy list accordingly, then subtracts off the ranking table entry from target and repeats.

Parameters:

  • rank – [in] The rank of the occupancy list to be computed

  • occ_list – [out] A pointer to an array of length nocc to store the occupancy list

  • rank_table – [in] Pointer to ranking table initialized by get_rank_table(uint64_t *table, size_t norb, size_t nocc)

  • norb – [in] Number of orbitals

  • nocc – [in] Number of occupancies; generally equal to the number of alpha or beta electrons in the system
