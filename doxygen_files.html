struct DoubleExcEntry
#include <hci_store.h>

A struct that stores the minimal amount of information necessary to compute a double excitation matrix element given the four changing orbitals encoded by rank (referred to as i, j, k, and l in ascending order).

[ik|lj]-[ij|lk] may be recovered by computing negative the sum of ijkl and iljk; all double excitation matrix elements involving these four orbitals may be mapped to one of these three values up to a sign (see get_double_exc_value_from_store(const DoubleExcEntry *exc_entry, const ExcResult *double_exc)).

Public Members

uint64_t rank

The combinatorial rank of the list of four changing orbitals; see rank_double_exc(size_t *exc_list, const ConfigInfo *config_info)

double ijkl

[ij|kl]-[il|kj] in chemist’s notation

double iljk

[il|jk]-[ik|jl] in chemist’s notation

struct MixedExcEntry
#include <hci_store.h>

A struct to represent mixed excitations.

Because the ERI tensor is real, only excitations with \(i<j\) and \(k<l\) need to be stored; there are no restrictions otherwise because \(i\) and \(j\) index \(\alpha\) MOs while \(k\) and \(l\) index \(\beta\) MOs.

static uint64_t rank(const size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

Get the combinatorial rank corresponding to the provided list of occupied orbitals.

Formula:

\[ \sum\limits_{i=1}^{N_\text{occ}}\binom{c_i}{i}, \]
where \(c_i\) is the \(i\)th orbital in the occupancy list sorted in ascending order

Parameters:

  • occ_list – [in] Pointer to an array of length \(N_\text{occ}\) specifying the zero-indexed occupied orbitals in ascending order

  • rank_table – [in] Pointer to ranking table initialized by load_rank_table(uint64_t *table, size_t norb, size_t nocc)

  • norb – [in] Size of orbital space

  • nocc – [in] Number of occupancies; generally equal to \(N_\alpha\) or \(N_\beta\)

Returns:

The rank of the specified occupancy list

static void unrank(uint64_t rank, size_t *occ_list, const uint64_t *rank_table, size_t norb, size_t nocc)

Greedy unranking algorithm for computing an occupancy list given its combinatorial rank.

Starting from the last row in the ranking table (corresponding to the highest occupied orbital), uses find_row_index(uint64_t target, uint64_t *row, size_t norb, size_t nocc) to find the column index of the entry that does not exceed the value of target, updates the occupancy list accordingly, then subtracts off the ranking table entry from target and repeats.

Parameters:

  • rank – [in] The rank of the occupancy list to be computed

  • occ_list – [out] Pointer to an array of length \(N_\text{occ}\) to store the occupancy list

  • rank_table – [in] Pointer to ranking table initialized by load_rank_table(uint64_t *table, size_t norb, size_t nocc)

  • norb – [in] Size of orbital space

  • nocc – [in] Number of occupancies; generally equal to \(N_\alpha\) or \(N_\beta\)

Initializes the \(N_\text{occ}\) by \(N_\text{orb}-N_\text{occ}+1\) table needed to rank and unrank combinations using the combinatorial number system.
