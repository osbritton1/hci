class hci.lib.hci_store.ERITensor(eri_mo_aaaa_s8, eri_mo_bbbb_s8, eri_mo_aabb_s4, config_info)[source]

Bases: object

Class storing the electron repulsion integrals. For a UHF starting point, three MO transformed tensors are needed of \([\alpha\alpha|\alpha\alpha]\), \([\alpha\alpha|\beta\beta]\), and \([\beta\beta|\beta\beta]\) type. When passed as an argument to a ctypes function, yields a reference to a struct compatible with the C Library’s ERITensor.

Parameters:

  • eri_mo_aaaa_s8 (ndarray[tuple[Any, ...], dtype[float64]]) – Eightfold compressed ERI tensor in the \(\alpha\) MO basis

  • eri_mo_bbbb_s8 (ndarray[tuple[Any, ...], dtype[float64]]) – Eightfold compressed ERI tensor in the \(\beta\) MO basis

  • eri_mo_aabb_s4 (ndarray[tuple[Any, ...], dtype[float64]]) – Fourfold compressed ERI tensor in the mixed MO basis

  • config_info (ConfigInfo) – ConfigInfo instance supplying ranking tables and key parameters of the system

hci.lib.hci_enlarge.enlarge_space_doubles(hcivec, add_thresh, config_info, exc_entries)[source]

Given a reference HCIVec, uses the HCI selection algorithm to generate a list of doubly excited deterinants to add to the configuration space.

Parameters:

  • hcivec (HCIVec) – Reference HCIVec tagged with list of ranks describing the current configuration space

  • add_thresh (float) – Selection threshold for adding a new determinant. A candidate determinant is added if the magnitude of its matrix element with a determinant from hcivec times the coefficient of that determinant exceeds add_thresh.

  • config_info (ConfigInfo) – ConfigInfo instance supplying ranking tables and key parameters of the system

  • exc_entries (ExcEntries) – ExcEntries object storing double excitations sorted desc. by magnitude

Raises:

  • HCIVec.RankNotFoundError – rank attribute of hcivec is missing, which is needed to define the current configuration space.

  • ExcEntries.SortTypeError – exc_entries has not been sorted in descending order by magnitude.

Returns:

A list of ranks corresponding to configurations that passed the selection threshold

Return type:

ndarray[tuple[Any, …], dtype[void]]
